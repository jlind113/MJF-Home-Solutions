# MJF Home Solutions

[![Netlify Status](https://api.netlify.com/api/v1/badges/193cce64-7adc-4d64-9e1c-20ff8510fc19/deploy-status)](https://app.netlify.com/projects/mjf-home-solutions/deploys)

A professional website for MJF Home Solutions, a Utah-based home improvement company specializing in drywall, carpentry, and flooring services.

## About

MJF Home Solutions provides premier drywall and construction services throughout the Bountiful, Ogden, Layton, and surrounding Utah areas. This website showcases our services, project gallery, and provides an easy way for customers to get in touch.

## Tech Stack

**Framework**: React Router 7
**Build Tool**: Vite
**UI Library**: Material-UI v7
**Styling**: Custom MUI Theme
**Deployment**: Netlify

## Features

**Responsive Design**
**Service Showcase**
**Project Gallery**
**Contact Form**
**Dark/Light Theme**
**SEO Optimized**
